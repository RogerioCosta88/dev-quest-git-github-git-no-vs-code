Git no Visual Studio Code

É possível fazer todo o processo do git dentro do próprio VS Code

Controle do Código Fonte

É usado para usar o Git diretamente no VSCode

Fica no menu da esquerda, logo após a lupa.

Git add

O git add . é realizado clicando no sinal de + que fica a direita do próprio arquivo quando o mouse é possicionado em cima desse arquivo.

Git commit

Após dar o git add . os arquivos estão prontos para serem comitados.

Para commitar escrevemos uma mensagem descrevendo o que foi alterado e/ou criado na caixa de mensagem logo acima da palavra Commit que também está acima dos arquivos recem adicionados e em seguida clicamos no simbolo de check que está acima da mensagem de commit.

Git push

Após commitar, os arquivos estão prontos para o push.

  1. Na mesma linha do sinal de check, tem ..., clique nesses três pontos

  2. Em seguida clique em Pull, Push

  3. E para finalizar o processo cliquen em Push

Ponto, todo o processo feito no Git Bash, foi realizado diretamente no VSCode
